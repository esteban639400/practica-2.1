Navegador web

Un navegador web(en ingles,web browser) es un software, aplicación o p rogrma que permite el acceso a la Web, interpretando la información de distintos tipos de archivos y sitios web para que estos puedan ser vistos.

La funcionalidad básica de un navegador web es permitir la visualización de documentos de texto, posiblemente con recursos multimedia incrustados

Ejemplos de navegadores web

SafariSafari es un navegador web de código cerrado desarrollado por Apple Inc.ESta disponible para macOS,iPadOS E iOS

Firefox es un navegador web de codigo cerrado desarrollado para Linux, Android,IOS,macOS y Microsoft Windows coordinado por la Corporación Mozilla y la fundación Mozilla.


Google Chrome Google Chrome es un navegador web de código cerrado desarrollado por Google,aunque derivado de proyecyos de código abierto
